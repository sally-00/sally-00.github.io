Wget is a software package for retrieving files using HTTP, HTTPS, FTP and FTPS
To download online files:
```bash
$ wget <the website>
```
To permit execution of files:
```bash
$ chmod +x <filename>
```

Add lines to a file:
```bash
$ echo “some text” > folder/file
```
